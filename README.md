# 萨卡兹文字转换器

《明日方舟：终末地》萨卡兹文字的加解密工具。

## 功能说明

**加密：** 完全可用。将中文文本通过模运算转换为萨卡兹字母串。

**解密：** 极度受限。解码器仅能匹配极少量游戏内专有词汇（角色名、地名等），基本无法翻译完整句子。

## 原理

每个汉字通过以下规则映射为一个字母：

```
Letter = REMAINDER_MAP[ord(汉字) % 56]
```

由于多个汉字可能共享同一余数，解码是一对多的——每个字母对应一组候选汉字，在没有完整游戏词典的情况下，还原完整句子几乎不可行。

## 使用方法

双击 `Sarkaz.bat` 运行。

无需安装任何第三方依赖。
